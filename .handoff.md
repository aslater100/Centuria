## 4. FAILED PATHS / GOTCHAS (save the next agent time)

- **Byte-identity hinges on `supplyShockSeverity() === 0` in healthy play.** That's true
  because severity is `(baselineHealth − supplyChainHealth)/baseline`, and healthy play keeps
  actual == the era-structural baseline (all raws flow). `0 × gain = 0` → no perturbation, so
  the full 926-test suite passes unchanged. If a future change makes healthy play dip supply
  health below baseline, this term (and the output drag) will fire — re-baseline if so.
- **It's a pure sink, deliberately.** Inflation feeds confidence/GDP but **never** sector
  output (`currencyEfficiency`/`economyOutputMult` don't read it), so supply→inflation can't
  loop back into the supply→output→raws path. That's why it's non-divergent. Don't wire
  inflation into sector output without re-checking this.
- **Placement & lag.** Cost-push lives in `tickMonetary` (central-bank-gated, line ~5229),
  which runs **before** `tickIntermediateGoods` (~5270) in the monthly tick — so it reads
  *last month's* cached `supplyChainHealth`. Intentional one-month price lag. A nation with no
  central bank gets no cost-push (no monetary dynamics exist there anyway).
- **The headless day/year mapping is compressed** — in `src/sim/headless.ts` an
  `OIL_EMBARGO_DAYS` (180) embargo can span ~3 *displayed* years because `r.day` advances
  slowly vs `r.year` in the fast sim. In real-time play 180 days = 6 months as intended. Don't
  read the headless calendar as wall-clock.
- **`npm install`:** `ELECTRON_SKIP_BINARY_DOWNLOAD=1 PLAYWRIGHT_SKIP_BROWSER_DOWNLOAD=1
  npm install --ignore-scripts --no-audit --no-fund` (electron's postinstall 403s in this web
  env and rolls back the whole install).
- **Asset/audio generation** still network-blocked (`huggingface.co` 403, no sharp/ffmpeg).
